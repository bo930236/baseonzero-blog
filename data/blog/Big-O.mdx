## Introduction

>  Talking about big O is talking about the worst scenario

- O(f(n)=n) means f(n) could be linear
- O(f(n)=n^2) means f(n) could be quadratic
- O(f(n)=1) means f(n) could be constant

## Counting Operations

- constant  f(n)=3   **O(1)**
```js
function addUpTo(n) {
	return n * ( n + 1 ) / 2
}
// 3 operations
```

- linear   f(n)=n   **O(n)**
```js
function addUpTo(n) {
	let total = 0
		for(i=1; i<=n; i++) {
			total += i        //loop causes more and more operations
		}
	return total
}
// n operations(n grows, more operations grow)
```

- 嵌套迴圈 O(f(n)=n^2)
```js
function printAllPairs(n) {
	for(var i = 0; i < n; i++) {
		for(var j = 0; i < n; j++) {
			console.log(i, j)
		}
	}
}
```
